# Social Network System Design

Проектирование социальной сети в рамках курса по System Design.

## Домашнее задание №2

Разработка REST API для социальной сети ВКонтакте.

## Реализованные операции

- добавление и удаление друзей
- просмотр друзей пользователя
- просмотр анкеты пользователя
- публикация поста
- загрузка медиа для постов
- просмотр ленты постов
- просмотр диалогов и чатов
- отправка и чтение сообщений

## API

OpenAPI-спецификация находится в файле:

```txt
api/rest_api.yml

1. Создать пост
POST /api/v1/posts

2. Получить конкретный пост
GET /api/v1/posts/{postId}

3. Загрузить медиа для поста
POST /api/v1/posts/{postId}/media

4. Получить домашнюю ленту пользователя
GET /api/v1/feed
